1. Для инициализации репозитория пишем: git init . (точка - текущая папка, можно указывать полный путь).

2. Чтобы понять текущее состояние репозитория пишем: git status.

3. Чтобы добавить файлы в staging area пишем: git add "имя файла или . (добавятся все измененные файлы из текущей папки)".

4. Чтобы сделать commit пишем: git commit -m "Описание коммита" (можно закоммитить и без "-m").

5. Чтобы посмотреть историю коммитов пишем: git log.
